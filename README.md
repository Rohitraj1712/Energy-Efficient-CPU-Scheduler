# Energy Efficient CPU Scheduling Algorithm

## Overview
This project focuses on designing an energy-efficient CPU scheduling algorithm using DVFS, thermal-aware scheduling, and workload prediction.

## Objectives
- Reduce energy consumption
- Maintain system performance
- Ensure fairness in scheduling

## Project Structure
- src/ → scheduling algorithms
- data/ → input data
- results/ → output graphs

## Tech Stack
- Python
- NumPy
- Matplotlib
